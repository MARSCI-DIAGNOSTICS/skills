Grok Code is xAI's coding-focused model. At $0.20 per million input tokens, it's the cheapest model in Cursor. Supports thinking mode and a 256k context window.
## Strengths
- $0.20/1M input tokens. The lowest-priced model available. Good for budget-conscious coding tasks and quick iterations.
- Optimized for fast code generation.
