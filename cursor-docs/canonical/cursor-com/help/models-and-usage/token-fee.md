# Cursor Token Fee

All Teams and Enterprise plan users are subject to a Cursor Token Fee of $0.25 per million tokens on all non-Auto agent requests. This applies to included usage, on-demand usage, and BYOK usage.

## What does the Cursor Token Fee cover?

- Semantic search and codebase indexing infrastructure
- Custom model execution and routing
- Processing and infrastructure costs

## How is the Cursor Token Fee calculated?

The fee applies to input tokens, output tokens, and cached tokens. It applies to all Teams and Enterprise plans, including BYOK usage. The fee is in addition to whatever you pay your API provider directly.

## How do I avoid the Cursor Token Fee?

Use the **Auto** model, which is exempt from the Cursor Token Fee.
