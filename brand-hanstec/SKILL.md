# Skill: HansTec Brand System

Al crear interfaces para HansTec, usar EXACTAMENTE estos valores:

## Colores (CSS Variables)
- Rojo principal: #E63946 (dark) / #C0392B (light)
- Rojo acento: #FF4757 (dark) / #E63946 (light)
- Negro base: #0A0A0A
- Crema: #F5F4F0
- Backgrounds dark: #0A0A0A, #111111, #1A1A1A
- Backgrounds light: #F5F4F0, #FFFFFF, #EDECE8

## Tipografía
- Bebas Neue: títulos, headings, números grandes, logos
- Rajdhani: texto de cuerpo, botones, descripciones
- Space Mono: etiquetas técnicas, labels, tags pequeños

## Logo SVG
La H de HansTec usa dos columnas verticales con una barra horizontal que tiene una diagonal de velocidad (brillo/shine).

## Tono visual
- Industrial/tech pero accesible
- Transiciones suaves (0.4s ease) en theme switch
- Líneas rojas como acento (bordes top, underlines)
- Sombras con red-glow en elementos destacados
- Grilla animada como textura de fondo
